## Windows 安装

1. 下载上面的 `.exe` 安装包。
2. 双击运行安装程序，按提示完成安装。
3. 从桌面或开始菜单启动 2KMAN。

如果 Windows 弹出安全提示，请确认安装包来自本仓库的正式 Release，再选择继续运行。

系统要求：Windows 10 或 Windows 11，64 位。

## 更新与问题反馈

客户端会检查稳定版本清单。发现新版本时，按提示下载并安装即可。中国大陆用户会自动使用可用的加速线路，其他地区使用 GitHub Release 直连。

如果遇到启动、安装或连接问题，请反馈：

- 使用的系统和版本；
- 2KMAN 客户端版本；
- 具体操作步骤和错误提示；
- 必要时附上相关截图。
