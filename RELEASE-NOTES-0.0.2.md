# 2KMAN 0.0.2

## Windows 更新

- 修复 Windows 打包后启动串流子进程时误触发“客户端已经在运行”。
- 串流子进程现在直接进入 Remote Play worker，不再创建第二个主界面或申请单实例锁。
- 安装包：`2KMAN-0.0.2-win-x64.exe`
- 类型：Windows x64 Inno Setup 安装程序
- 最低系统：Windows 10 x64
- SHA-256：`7FFAE1FD1C140DEE9D6512C18C993FB737580B2B858B644D894E215697AFA42B`

## 发布线路

- 中国大陆：客户端自动使用 GitHub 加速代理。
- 其他地区：客户端使用 GitHub Release 直连。
- macOS 暂无可用安装包。
