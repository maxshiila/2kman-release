# 2KMAN 0.0.2 更新说明

## 这次更新了什么

- 修复了 Windows 版本启动后，连接游戏画面时偶尔提示“客户端已经在运行”的问题。
- 改善了进入比赛、进攻、防守和投篮过程中的自动操作，让动作衔接更加稳定。
- 改善了比赛结束后的识别和返回流程，减少卡在页面或过早退出的情况。
- 优化了画面识别和等待时机，减少因为画面短暂变化导致的误判。

## 下载和安装

- Windows 10/11（64 位）：[下载 2KMAN 0.0.2](https://github.com/maxshiila/2kman-release/releases/download/v0.0.2/2KMAN-0.0.2-win-x64.exe)
- 下载后双击安装包，按照提示完成安装即可。
- macOS 版本暂未提供。
