# 2KMAN 0.1.0 更新说明

## 这次更新了什么

- 增加 asset1 点击后的画面变化确认，画面未变化时最多自动重试一次，避免连续误按。
- 假传后等待投篮调整为 0.7 秒。
- 回调到下一次投篮命中前，真实传球只发送一次。
- 监视器将进攻、防守、投篮、篮板状态显示在顶部信息栏，移除多余提示文案。
- 保持实时进度设备链接在会话重置后稳定可用。

## 下载和安装

- Windows 10/11（64 位）：[下载 2KMAN 0.1.0](https://github.com/maxshiila/2kman-release/releases/download/v0.1.0/2KMAN-0.1.0-win-x64.exe)
- 下载后双击安装包，按照提示完成安装即可。
- macOS 版本暂未提供。
