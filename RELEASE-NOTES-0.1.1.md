# 2KMAN 0.1.1 更新说明

## 这次更新了什么

- 报错日志现在保留状态机的真实失败原因，不再只显示“自动化运行时失败”。
- 自动化报错后，主界面可以直接点击“开始”重新连接和运行，不需要先关闭客户端。
- 保留 0.1.0 的 asset1 画面变化确认、单次重试、0.7 秒假传等待和单次回传限制。

## 下载和安装

- Windows 10/11（64 位）：[下载 2KMAN 0.1.1](https://github.com/maxshiila/2kman-release/releases/download/v0.1.1/2KMAN-0.1.1-win-x64.exe)
- 下载后双击安装包，按照提示完成安装即可。
- macOS 版本暂未提供。
