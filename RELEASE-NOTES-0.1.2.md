# 2KMAN 0.1.2 更新说明

## 这次更新了什么

- 未激活正式授权时，查找和配对 PS5 会在后端直接拒绝。
- 修复手动串联在监视器窗口获得焦点后键盘输入无反应的问题。
- 手动串联不再显示自动攻防、投篮和篮板状态，自动/手动串联切换保持可用。
- 检查到新版本时只弹出提示，不会自动下载；下载必须由用户明确点击。
- 保留 asset1 画面变化确认、单次重试、0.7 秒假传等待和回传去重逻辑。

## 下载和安装

- Windows 10/11（64 位）：[下载 2KMAN 0.1.2](https://github.com/maxshiila/2kman-release/releases/download/v0.1.2/2KMAN-0.1.2-win-x64.exe)
- 下载后双击安装包，按照提示完成安装即可。
- macOS 版本暂未提供。
