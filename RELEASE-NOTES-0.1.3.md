# 2KMAN 0.1.3 更新说明

## 这次更新了什么

- 主界面新增“实时进度”入口，复制设备专属链接并用系统浏览器打开 Monitor 页面。
- 修复带逗号金币数值的读取与显示。
- 延续 0.1.2 的授权、手动串联、更新提示和比赛流程修复。

## 下载和安装

- Windows 10/11（64 位）：[下载 2KMAN 0.1.3](https://github.com/maxshiila/2kman-release/releases/download/v0.1.3/2KMAN-0.1.3-win-x64.exe)
- 下载后双击安装包，按照提示完成安装即可。
- macOS 版本暂未提供。
