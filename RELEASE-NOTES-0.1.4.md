# 2KMAN 0.1.4 更新说明

## 这次更新了什么

- 修复 asset1/开放球场等自动化识别失败时只停不重连的问题。
- 自动化失败后立即断开并尝试重连；首次失败后在 5 分钟窗口内持续重试，超时仍未连上才停止。
- 错误日志记录具体失败阶段和原因。

## 下载和安装

- Windows 10/11（64 位）：[下载 2KMAN 0.1.4](https://github.com/maxshiila/2kman-release/releases/download/v0.1.4/2KMAN-0.1.4-win-x64.exe)
- 下载后双击安装包，按照提示完成安装即可。
- macOS 版本暂未提供。
